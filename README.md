# job4j_todo
**Общее описание**

Этот учебный проект реализует УПРОЩЁННЫЙ сайт "список задач".

**Использованы технологии:**

Java 17,<br/>
Spring Boot 3.4,<br/>
Thymeleaf 3.4,<br/>
Bootstrap 4.1.3,<br/>
Liquibase 4.28.0,<br/>
Hibernate 5.6.11.Final,<br/>
Lombok 1.18.22,<br/>
PostgreSQL 16,<br/>
Junit 5.<br/>

**Требования к окружению. (Список ПО, которое должно быть установлено для запуска проекта):**

Java 17, <br/>
Maven 3.8, <br/>
PostgreSQL 14<br/>

**Техническое задание**

Представления:

1. Страница со списком всех заданий. В таблице отображаем имя(короткое описание), дату создания и состояние (выполнено или нет);
    - На странице со списком добавить кнопку "Добавить задание".
    - На странице со списком должно быть три ссылки: "Все", "Выполненные", "Новые". При переходе по ссылкам в таблице нужно отображать: все задания, только выполненные или только новые.
2. Страница с подробным описанием задания;
    - При клике на задание переходим на страницу с подробным описанием задания.
    - На странице с подробным описанием добавить кнопки: "Выполнено", "Редактировать", "Удалить".
    - Если нажали на кнопку выполнить, то задание переводится в состояние выполнено.
    - Кнопка "Редактировать" переводит пользователя на отдельную страницу для редактирования.
    - Кнопка "Удалить" удаляет задание и переходит на список всех заданий.
3. Страница для редактирования задания;
